# Zirr - Universal IR TV Remote

Zirr is a universal infrared (IR) remote control application designed for Android devices.
It allows users to effortlessly control their televisions directly from their smartphones equipped
with a built-in IR blaster.

## Features
- **Universal TV Support**: Works with a wide variety of television brands and models.
- **Offline Functionality**: Requires no internet connection to send IR commands.
- **Easy to Use**: Clean, simple, and user-friendly interface.
